# 💫 About Me:
🚀 Computer Science student from Denmark and currently working as a Software Developer with a strong focus on full-stack web development.<br><br>I started seriously pursuing development in May 2025 by putting in extensive interest hours alongside a self-employed full-stack developer, which led to being hired in March 2026 after proving my skills and dedication.<br><br>💻 My core stack is C#, .NET, React, and TanStack, with modern frontend development fully replacing Razor Pages in my workflow.<br><br>⚡ I enjoy building practical, scalable systems that solve real problems—like BarShifts, a management platform for school bars that replaces messy Google Drive docs and Messenger chats with structured team management, event planning, and task assignment.<br><br>🎮 Outside of web development, I’m also exploring Unity and working toward building my own 2D game projects.<br><br>I care about clean architecture, strong fundamentals, and understanding why solutions work—not just making them work.
